## 🛠️ Development

```bash
# Clone repository
git clone https://github.com/Lesnak1/monad-synapse.git
cd monad-synapse

# Install dependencies  
cd apps/web && npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```

**No environment variables needed!** The app works immediately with secure fallbacks.
